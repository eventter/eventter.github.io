EventterMQ

Distributed message queue & streaming server

Get Started View on GitHub

Fast Messaging

Supports multiple standard messaging protocols, reliable delivery, message acknowledgements & flexible routing.

Reliable Stream Data Storage

Persist stream data (e.g. user events, metrics, logs) to be accessed, searched & processed later with configurable retention.

Distributed & Fault-tolerant

Built from the beginning to be run as a cluster that can tolerate failure of minority of the servers by using Raft consensus algorithm & replication.
